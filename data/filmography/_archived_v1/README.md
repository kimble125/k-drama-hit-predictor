# filmography/_archived_v1/

v3 정정 과정에서 사용 중지된 필모그래피 파일 보관소.

## 보관 사유

`candidate_dramas.csv`의 v1/v2 시점에 잘못 매칭됐던 인물들의 필모그래피. v3.1에서 다음 정정으로 더 이상 참조되지 않음:

### 잘못된 감독 매칭 (5건)
- **박현석.json**: v2 CSV에 골드랜드·"모두가 자신의…" 감독으로 기재됐으나 정정 → 김성훈(골드랜드) / 차영훈(모두가)
- **신경수.json**: v1 CSV에 허수아비 감독 → 정정 → 박준우
- **이충현.json**: v1 CSV에 기리고 감독 → 정정 → 박윤서
- **조영민.json**: v2 CSV에 "오늘도 매진했습니다" 감독 → v3 자동수집 결과 → 이슬기 (재검증 필요)
- **김종연.json**: v1 시점 매칭, 출처 불명

### 잘못된 작가 매칭 (7건)
- **임상춘.json**: 골드랜드 → 정정 → 황조윤
- **정보훈.json**: "모두가 자신의…" → 정정 → 박해영
- **박경수.json**: 허수아비 → 정정 → 이지현
- **이영미.json**: "오늘도 매진했습니다" → 자동수집 → 진승희
- **이충현.json**: 기리고 → 정정 → 박중섭
- **김도하·김우리.json**: v1 시점 매칭, 출처 불명

### 잘못된 캐스트 매칭 (10건)
유미의 세포들3 / 골드랜드 / 21세기 대군부인의 v2 시점 잘못 기재된 캐스트의 필모그래피. v3 정정으로 캐스트가 교체되어 더 이상 참조되지 않음.

- **유미의 세포들3 v2 잘못된 캐스트**: 박진영, 진영, 원진아 (정답: 김재원, 전석호, 최다니엘)
- **골드랜드 v2 잘못된 캐스트**: 정해인, 조진웅, 염혜란 (정답: 박보영·김성철·이현욱·김희원·문정희·이광수)
- **21세기 대군부인 v2 시점 표기**: 이지아, 성동일 (현 캐스트는 노상현·공승연 추가)
- **안종연·김원해**: v2 CSV 매칭, 출처 불명

## 활용

만약 향후 다른 드라마에서 동명이인이 잡히면 데이터 검증 자료로 참고 가능. 단, **자동 RSI 계산엔 사용하지 않음**.
